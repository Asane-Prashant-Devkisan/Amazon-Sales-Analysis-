📊 Amazon Sales Analysis

Turning raw data into actionable insights using Python, SQL, and Power BI.

📌 Project Overview

This project presents an end-to-end data analysis workflow on an Amazon e-commerce dataset. The dataset, containing over 129,000 records, was synthetically generated using Python to simulate real-world online retail scenarios.
The project demonstrates key data analytics skills including data generation, exploratory data analysis (EDA), SQL querying, and dashboard visualization. The goal is to extract meaningful insights related to sales trends, customer behavior, and product performance.

🎯 Objectives
Analyze sales trends over time
Identify top-performing product categories
Understand regional sales distribution
Evaluate the impact of promotions on sales
Analyze order status and fulfilment efficiency
🛠️ Tech Stack
Python (Pandas, NumPy, Matplotlib, Seaborn)
SQL (Data extraction & analysis)
Power BI (Dashboard & visualization)
📂 Dataset Information
📦 Total Records: 129,000+
📅 Time Range: 2022 – 2024
🌍 Geography: Multiple states in India
🧾 Data Type: Synthetic (generated using Python)
Key Features:
Order details (ID, date, status)
Product details (category, size, SKU)
Pricing & quantity
Customer location
Promotions & discounts
Shipping & fulfilment
🐍 Dataset Generation

The dataset was created using a custom Python script that simulates real-world e-commerce behavior.

Key Highlights:

Regional demand variation
